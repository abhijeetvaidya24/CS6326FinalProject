Instructions to run our code -

1. Download the source code from the github link provided in the submission.

2. Extract the zip file downloaded.

3. Find and open demo.ipynb file on Google Colab.

4. Run every command cell

5. Select the Image and the GIF Video on which you want to test the Image Animation alorithm.

6. You can select the images and videos from the list provided or you ca upload them.

7. The test images that we used are stored in the 'test_images' folder. You can upload these images to check our results mentioned in the project report.

8. Select the training algorithm/dataset from the list.

9. Click Generate and wait for the results.
